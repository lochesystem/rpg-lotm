# Senhor dos Mistérios — RPG de Mesa

Sistema de RPG de mesa fan-made baseado em **Lord of the Mysteries** (*Senhor dos Mistérios*), com regras híbridas inspiradas em Vampiro: A Máscara e D&D (d20).

## Conteúdo

- **Livro do Jogador** — regras, 22 Caminhos, Sequências 9–5, combate, misticismo
- **Livro do Mestre** — lore, organizações, NPCs, bestiário, artefatos selados; apêndice [Guia dos 22 Caminhos](livro-mestre/guia-22-caminhos-deuses.html)
- **Livro de Campanhas** — [mini-campanha para iniciantes](campanhas/00-patrulha-noturna.html) (2–3 jogadores, 3–4 sessões) + 3 campanhas longas + [tutorial sessão 1](campanhas/exemplo-sessao-iniciantes.html)
- **Apêndices** — [guia de criação de ficha](apendices/guia-criacao-ficha.html), ficha de personagem, [exemplo Leitor Seq. 9](apendices/ficha-exemplo-leitor-seq9.html), tabelas, glossário

## Uso local

Abra `index.html` no navegador ou sirva a pasta com qualquer servidor estático.

Para regenerar páginas HTML a partir dos `.md` (guia, tutorial, continuidade): `npm run build:pages`
